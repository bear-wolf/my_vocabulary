# my_vocabulary

## Client
- Start application on port: 4200
    - > npm run start

## Server
- migration
    - > cd ./server/migration
    - > npm run up

* Data migration is currently not done via seed, but is designed as a normal migration, because something is wrong with local packages.

- db PostgreSQL
    - > cd ./server/
    - > npm run start.db

- run server on port 4000
    - > npm run start
